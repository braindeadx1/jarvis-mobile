# Jarvis Notify

Sendet eingehende WhatsApp/Discord-Nachrichten an den Jarvis Mobile Server,
damit Jarvis sie vorlesen oder anzeigen kann.

## Trigger
Dieser Skill wird automatisch bei eingehenden Nachrichten aufgerufen,
wenn in TOOLS.md entsprechend konfiguriert.

## Konfiguration
Die Jarvis-Server-URL und das Webhook-Secret werden im Script konfiguriert.
